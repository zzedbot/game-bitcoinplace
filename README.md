# BitcoinPlace ⚡

> 一个结合比特币经济模型的协作画布游戏 —— 2100 万像素 = 2100 万枚比特币

## 🎯 项目愿景

将 r/place 的协作画布玩法与比特币的经济模型结合，创造一个去中心化、稀缺性驱动的数字艺术实验。

## 📊 核心概念

| 比特币 | BitcoinPlace |
|---|---|
| 总量 2100 万 BTC | 2100 万个坐标点 (7000×3000) |
| 21 万个区块 | 21 个矿区 (每区 1000×1000) |
| 区块时间 ~10 分钟 | 色块解锁周期 |
| 减半周期 4 年 | 可配置 (7 天/30 天/自定义) |
| 挖矿奖励 | 在线用户随机获得染色权 |
| 钱包 | 用户账户 + 库存系统 |
| 交易 | 拍卖行买卖染色权 |

## 🎮 核心玩法

1. **获得染色权**：用户在线即可参与随机分配，无需挖矿操作
2. **染色**：使用染色权为透明坐标上色（16 色调色板）
3. **修改**：拥有者可随时修改自己坐标的颜色
4. **拍卖**：未使用的染色权可挂单拍卖
5. **收藏**：已使用的染色权不可拍卖，但可永久修改
